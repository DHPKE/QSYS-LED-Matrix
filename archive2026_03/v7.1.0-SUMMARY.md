## Technical Notes

### Q-SYS Plugin Tab System
- `GetPages()` returns an array of tab names
- Q-SYS automatically creates tabs and manages `props["page_index"]`
- Page index is zero-based (0-4 for 5 tabs)
- Controls not in current page's layout are hidden but maintain state

### Lua Pattern Used
```lua
function GetControlLayout(props)
    local currentPage = props["page_index"].Value
    
    if currentPage == 0 then
        -- Page 0 layout
    elseif currentPage == 1 then
        -- Page 1 layout
    elseif currentPage == 2 then
        -- Page 2 layout
    elseif currentPage == 3 then
        -- Page 3 layout
    elseif currentPage == 4 then
        -- Page 4 layout
    end
    
    return layout, graphics
end
```

This pattern ensures clean separation and easy maintenance.
